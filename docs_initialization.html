The TopcoatTouch is instantiated with the TopcoatTouch function. var tt = new TopcoatTouch([container], [options]); Where container is query object (we will use the generic term query object for jQuery or Zepto query results) that is the base container where you wish the TopcoatTouch environment to be initialized. Almost always this is the $('body'), which is the default if no container is passed, but it can be any container (for testing, for example).

Configurable options

exceptionOnError
default false
Whether or not an exception is thrown if an error occurs. By default only a console error is emitted
initializeFunction
default none
Override the default template initialization function. See
menu
default false
Set to true if you want to set the menu on each page, or add items as a JSON array that will appear on every page. Eg: [{id: 'help', name: 'Help'}, {id: 'info', name: 'Info'}, {id: 'about', name: 'About'}]
menuFadeIn
default 100
Speed to fade in the menu.
menuFadeOut
default 50
Speed to fade out the menu.
menuHasIcons
default false
renderFunction
default none
templateDirectory
default 'templates'
