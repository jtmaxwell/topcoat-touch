TopcoatTouch Additions

The following features are additions to Topcoat through the TopcoatTouch library.

Scrollable

Add the "scrollable" class to one container on a page to make that container scrollable through IScroll. By default the document scrolls vertically, if you want it to scroll in different directions add set the data-scroll-x and data-scroll-y properties.
Example:
<div class="content scrollable" data-scroll-x="false" data-scroll-y="true"> <div> -- Scrollable content goes in here -- </div> </div>

List View

Create a div with the class "topcoat-list__container" and place an unordered list within it with the class "topcoat-list". The list items should have the class "topcoat-list__item". Note: Adding a span with the class "chevron" produces a chevron which indicates that the list item is actually a link to a different page.
Example:
<div class="topcoat-list__container"> <ul id="demoList" class="topcoat-list list"> <!-- setting the data-rel means that when that element is clicked it goes to the page in the data-rel --> <li class="topcoat-list__item" data-rel="buttonExample"> <span class="formItem">Buttons</span><span class="chevron"></span><span class="count"></span> </li> <li class="topcoat-list__item" data-rel="checkRadioExample"> <span class="formItem">Check/Radio Example</span><span class="chevron"></span><span class="count"></span> </li> <li class="topcoat-list__item" data-rel="formExample"> <span class="formItem">Form Example</span><span class="chevron"></span><span class="count"></span> </li> <li class="topcoat-list__item" data-rel="waitingDialogExample"> <span class="formItem">Waiting/Dialog Example</span><span class="chevron"></span><span class="count"></span> </li> <li class="topcoat-list__item" data-rel="galleryExample"> <span class="formItem">Gallery Example</span><span class="chevron"></span><span class="count"></span> </li> <li class="topcoat-list__item" data-rel="carouselExample"> <span class="formItem">Carousel Example</span><span class="chevron"></span><span class="count"></span> </li> </ul> </div>

Drop Down

Create a dropdown with a div of class "form-dropdown". Within that div should be an anchor of class "toggle-dropdown" and an unordered list with the class "dropdown". The list items should be of class "dropdown-item" and you want to automatically have data that is presented on the change event add a data-id to the anchor. The list item should contain a div of class "menu-item"
Example:
<div class="form-dropdown" id="gallery-picker"><!-- To see how dropdowns are created go to --> <a href="#" class="toggle-dropdown">Gallery Type</a> <ul id="gallery-type" class="dropdown"> <li class="dropdown-item" data-id="kittens"> <div class="menu-item">Kittens</div> </li> <li class="dropdown-item" data-id="bears"> <div class="menu-item">Bears</div> </li> ... </ul> </div>
You react to the dropdown in the following way
$('#gallery-picker').change(function(e, id) { console.log('The new id is: ' + id); });

Menu

The menu requires some either on the html or in a template (if you are building an MVC project), as well as either a menu passed in as an initialization object or handling the SHOW_MENU and adjusting the menu array from there.
Step 1 is adding the menu to the navigation bar.
<div class="topcoat-navigation-bar"> <!-- Navigation bar --> <div class="topcoat-navigation-bar__item left quarter"> </div> <div class="topcoat-navigation-bar__item center half"> <h1 class="topcoat-navigation-bar__title">Topcoat Touch</h1> </div> <div class="topcoat-navigation-bar__item right quarter"> <a class="topcoat-icon-button--quiet menu-button" href="#"> <span class="topcoat-icon topcoat-icon--menu-stack"></span> </a> </div> </div>
While in the navigation bar, the other classes affect the appearance, the only class actually required for a menu is the menu-button class. Inside your application script you then either have to initialize the menu (and it will be included in all menu instances)
Example menu through initializing:
var tt = new TopcoatTouch({menu: [{id: 'help', name: 'Help'}, {id: 'info', name: 'Info'}, {id: 'about', name: 'About'}]});
Example menu through adding to the menuItems in the MENU_SHOW event:
tt.on(tt.EVENTS.SHOW_MENU, function(currentPage, menuItems) { if (currentPage != 'about')) { menuItems.push({name: 'About', 'id': 'about'}); } return menuItems; // Since menuItems isn't created in the function it is optional to return the array... });
Example menu through adding to the menuItems in the MENU_SHOW event:
tt.on(tt.EVENTS.SHOW_MENU, function(currentPage, menuItems) { var menuItems = {menu: [{id: 'help', name: 'Help'}, {id: 'info', name: 'Info'}, {id: 'about', name: 'About'}]}; return menuItems; // Since menuItems is created in the function it is required to return the array... });

Side Drawer

The side drawer is added to the bottom of the page and given the class "side-drawer". It ias automatically shown when the page is swiped to the right, and hidden when the page is swipped to the left (requires the hammer.js library to be included in the project). You can optionally add a button with the class "side-drawer-toggle" that can be clicked to show or hide the side-drawer, or you can manually show and hide the drawer by calling the method showSideDrawer() and hideSideDrawer() on the topcoat touch object.
Step 1: A side drawer container.
<div class="side-drawer"> <div class="topcoat-list__container"> <ul class="topcoat-list"> <li class="topcoat-list__item"> <span class="formItem">List Item 1</span> </li> <li class="topcoat-list__item"> <span class="formItem">List Item 2</span> </li> <li class="topcoat-list__item"> <span class="formItem">List Item 3</span> </li> <li class="topcoat-list__item"> <span class="formItem">List Item 4</span> </li> <li class="topcoat-list__item"> <span class="formItem">List Item 5</span> </li> <li class="topcoat-list__item"> <span class="formItem">List Item 6</span> </li> </ul> </div> </div>
Step 2 (Optional): Add a button to show hide the side draw in the navigation header.
<div class="topcoat-navigation-bar__item left quarter"> <a class="topcoat-icon-button--quiet side-drawer-toggle" href="#"> <span class="topcoat-icon topcoat-icon--menu-stack"></span> </a> </div>
Step 2 b (Optional): show or hide the side-drawer.
tt.showSideDrawer(); ... // Code Here tt.hideSideDrawer();
